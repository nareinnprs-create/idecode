## Website

Visit [https://opraiz.ai](https://opraiz.ai) for more information about OPRAIZ IDECODE.

## Repositories

This is the main repository for the OPRAIZ IDECODE project, containing the sources of the IDE platform. Please open generic discussions, bug reports and feature requests about OPRAIZ IDECODE on this repository. This project is a fork of [Eclipse Theia](https://github.com/eclipse-theia/theia).

## Releases

- [All available releases](https://github.com/opraiz/idecode/releases) are available on GitHub including changelogs.

## Scope

- Support building browser-based and desktop IDEs and tools
- Provide a highly flexible architecture for adopters
- Support VS Code Extension protocol
- Develop as open-source software

## Getting Started

Here you can find guides and examples for common scenarios to adopt OPRAIZ IDECODE:

- Fork the repository and [run the examples from source](doc/Developing.md#quick-start)
- Get familiar with [the development workflow](doc/Developing.md) and [Coding Guidelines](doc/coding-guidelines.md)

## Contributing

Read below to learn how to take part in improving OPRAIZ IDECODE:

- Fork the repository and [run the examples from source](doc/Developing.md#quick-start)
- Get familiar with [the development workflow](doc/Developing.md), [Coding Guidelines](doc/coding-guidelines.md), and [Code of Conduct](CODE_OF_CONDUCT.md)
- Find an issue to work on and submit a pull request
- Could not find an issue? Look for bugs, typos, and missing features.

## Feedback

- Join the discussion on [GitHub](https://github.com/opraiz/idecode/discussions).
- Ask a question, request a new feature and file a bug with [GitHub issues](https://github.com/opraiz/idecode/issues/new/choose).
- Star the repository to show your support.
