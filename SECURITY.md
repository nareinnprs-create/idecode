# OPRAIZ IDECODE Vulnerability Reporting Policy

If you think or suspect that you have discovered a new security vulnerability
in this project, please __do not__ disclose it on GitHub, e.g. in an issue, a
PR, or a discussion. Any such disclosure will be removed/deleted on sight.

Instead, please report any potential vulnerability by emailing the OPRAIZ
security team at **security@opraiz.ai**. Make sure to provide a concise
description of the issue, a CWE, and other supporting information.
